# init-ts

An ~~un~~opinionated batteries included TypeScript project generator.

### Usage

```bash
npx @xetrics/init-ts [options]

Options:
  -h, --help     Show help                                                 [boolean]
  -n, --name     Name of your project                                       [string]
  -v, --verbose  Enables verbose logging                                   [boolean]
```
